# Nhom3_ASM_Quanlikhachsan

## Giới thiệu

Dự án được thực hiện trong môn **Kỹ thuật Phần mềm** nhằm xây dựng hệ thống quản lý khách sạn hỗ trợ quản lý phòng, khách hàng, đặt phòng và thanh toán.

## Thành viên nhóm

* Tạ Minh Hoàng Khang
* Nguyễn Ngọc Minh Anh
* Trương Tấn Dũng
* Giáp Quốc An
* Hoàng Long Lâm
* Vũ Ngọc Thắng

## Chức năng chính

* Đăng nhập, đăng ký
* Quản lý phòng
* Đặt phòng
* Check-in
* Check-out
* Thanh toán
* Quản lý khách hàng
* Báo cáo doanh thu

## Công nghệ sử dụng

* SQL Server

* GitHub
